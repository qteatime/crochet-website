Crochet for Functional Programmers
==================================

Hello, friend.

This guide will work you through the Crochet programming language.
Through an example-driven approach, you will learn the big ideas
behind Crochet, the patterns and philosophy that drive how to
approach design problems, and other useful tidbits.

Because this guide is neither project nor domain focused, we will
assume that you have some knowledge of programming---and
functional programming in particular. This means that many of the
basic and common concepts that happen in Crochet won't be explained in
this guide, and there will be no directions on how to apply these
concepts to build specific games or applications.
