A language for secure applications and games.

Using Crochet, you can create, run, and remix applications while worrying less about privacy and security. Capabilities ensure that code can only use your computer and data in ways you know and consent to.

Crochet is free to use, open-source, and anything you create with Crochet can be used for commercial purposes.

Install it with npm
$ npm install @qteatime/crochet@experimental
