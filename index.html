Crochet is a tool for creating and remixing interactive stories, safely.

Create, run, and remix content worrying less about your privacy and security. Content built with Crochet can only use your computer and data in ways you know and consent to.

Crochet is free to use, open-source, and anything you create with Crochet can be used for commercial purposes.

Install it with npm
$ npm install @qteatime/crochet@experimental
